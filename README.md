# 1. Udemy_ React 강의

## 1. 강의수강 이유
1. 제5회 오픈 인프라 개발 경진대회 (OIDC 2023)
2. 2023 제4회 K-디지털 트레이닝 HACKATHON 디지털 융복합 시대의 디지털 혁신 서비스 개발 해커톤

프론트엔드 담당으로 React.js와 Material UI를 사용하여 참가.
개인공부로 역량을 강화하기 위함.
